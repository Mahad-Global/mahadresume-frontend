# MahadResume Frontend

React SPA for MahadResume — AI-powered resume builder for the UAE/Gulf market.

**Stack:** React 18, Vite, Tailwind CSS, Zustand, React Hook Form, Axios

**Backend repo:** https://github.com/Mahad-Global/mahadresume-backend

## Pages

- Landing, Login, Register, Forgot Password
- Dashboard: Resume Builder, ATS Checker, Job Match, Cover Letter, Templates, Subscription
- Employer: Dashboard, Campaigns, Applications, Candidate Search
- Campaign Apply (public invite link)

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build    # Output: dist/
```

## Environment

```
VITE_API_URL=http://127.0.0.1:8000/api/v1
VITE_APP_NAME=MahadResume
```
